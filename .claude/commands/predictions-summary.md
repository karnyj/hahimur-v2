סכם את ניחושי הקבוצה למשחק נתון. קבל מזהה משחק כארגומנט (למשל `B2`, `D2`).

## שלבים

### 1. שלוף את כל הניחושים

```bash
grep -r "$ARGUMENTS" src/users/ | grep "id: '$ARGUMENTS'"
```

### 2. חשב

מתוך הפלט, חשב עבור כל 26 המשתתפים:
- סה"כ שערים לקבוצת הבית
- סה"כ שערים לקבוצת החוץ
- כמה ניבאו ניצחון בית / תיקו / ניצחון חוץ
- האם יש עובדות בולטות: פה אחד לתוצאה אחת, קבוצה שאף אחד לא ניבא לה שערים, וכו׳

### 3. הצג בעברית

עטוף את כל הפלט בתוך code block (כך שיופיע כפתור copy):

```
⚽ [שם בית] נגד [שם חוץ]

[בית]: X שערים סה"כ
[חוץ]: X שערים סה"כ

פילוח התוצאות:
ניצחון [בית]: X אנשים.
תיקו: X אנשים.
ניצחון [חוץ]: X אנשים.
```

כללי פורמט:
- 0 אנשים → "אף אחד" (ללא נקודה)
- תוצאות עם 5 אנשים או פחות → הוסף בסוגריים אחרי המספר, אנשים עם אותה תוצאה מקובצים עם /: `(עודד 0-0, מאור/טל/אייל/תומר 1-1, נועם בירן 2-2)`
- שמות פרטיים בלבד. אם יש שני אנשים עם אותו שם פרטי, הוסף את ראשית שם המשפחה
- תוצאות בפורמט X-X (מקף, לא נקודותיים)
